# Online Bookstore App

### Version

0.1.0

### About The Project

This project is created for an online bookstore web application.
The REST API for the mentioned web application was coded using technologies such as

- Java
- Spring boot
- JPA hibernate
- Lombok
- JWT tokens
- MySQL
  <br>
